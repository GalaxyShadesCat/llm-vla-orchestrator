# llm-vla-orchestrator

Hierarchical multimodal LLM → VLA orchestration framework integrating GPT models with π₀ (openpi) for embodied robotics research.

## Python Version

Python 3.11 (managed via `uv`)

## Installation

Clone the repository:

```bash
git clone --recurse-submodules https://github.com/GalaxyShadesCat/llm-vla-orchestrator
cd llm-vla-orchestrator
````

Run setup:

```bash
bash setup.sh
```

Activate the environment:

```bash
source .venv/bin/activate
```

Or run directly:

```bash
uv run --python .venv/bin/python python your_script.py
```
